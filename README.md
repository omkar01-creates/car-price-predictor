# Car Price Predictor

This is a simple machine learning project where I built a web app using Streamlit to predict the price of a car. The user can select details like insurance_type, ownership, fuel_type, transmission, and kilometers driven, and the app gives an prediction of the car price based on the ML linear regression and multiple regression model.

## What this project does

The app takes some basic car details from the user and uses a trained machine learning model to predict the price of the car in lakhs. It also shows a small chart to give a rough idea of how different inputs affect the result.

## Features

- Simple and clean UI
- Dropdown options instead of typing inputs
- Instant price prediction
- Small chart shown below the result
- Dark theme layout

## Tech used

- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib
- Pickle
